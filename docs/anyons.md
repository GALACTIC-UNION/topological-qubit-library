# Anyon Model Catalogue

Documentation for each anyon model implemented in `tql.anyons`.

## Ising Anyons

- **Topological charge:** {1, σ, ψ}
- **Quantum dimension:** d_σ = √2
- **Fusion rules:** σ × σ = 1 + ψ
- **Physical candidates:** Majorana zero modes in topological superconductors, ν=5/2 FQH state

## Fibonacci Anyons

- **Topological charge:** {1, τ}
- **Quantum dimension:** d_τ = φ = (1+√5)/2 (golden ratio)
- **Fusion rules:** τ × τ = 1 + τ
- **Universal:** Yes — braiding alone is sufficient for universal QC
- **Physical candidates:** ν=12/5 FQH state (theoretical)
